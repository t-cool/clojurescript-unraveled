## 出版の方針

・Clojure(Script) の普及を目的とするプロジェクトです。

・Clojurescript Unraveled の改訂に合わせて、翻訳版をアップデートします。

・電子版を無料で配布します。また、印刷資金を募り、物理本を各種イベントで配布します。
